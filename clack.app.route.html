NAME

Clack.App.Route - URL dispatcher.

SYNOPSIS

(defpackage clack-sample  
  (:use :cl  
        :clack  
        :clack.app.route))  
(in-package :clack-sample)  
 
(defroute app (req)  
  (GET "/" #'index)  
  (GET "/login" #'login)  
  (POST "/login" #'authorize)  
  (GET "/member/:id" #'member))  
 
(clackup #'app)

DESCRIPTION

Clack.App.Route provides an URL based dispacher, inspired by Ruby's Sinatra.
