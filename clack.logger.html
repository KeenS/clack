NAME

Clack.Logger - Logging utility for Clack.

AUTHOR

Eitarow Fukamachi (e.arrows@gmail.com)

SEE ALSO

EXTERNAL SYMBOLS

  • Variable: *logger-output*

  • Variable: *logger-time-format*

  • Variable: +debug+

  • Variable: +info+

  • Variable: +notice+

  • Variable: +warning+

  • Variable: +error+

  • Variable: +critical+

  • Variable: +alert+

  • Variable: +emergency+

  • Variable: *logger-min-level*

  • Function: log-message [level message]

    • Output a log if the log level is more than `logger-min-level'. Log level must be a integer 0-7, or a keyword represents log level.

    Example: (log-message :warning "Something wrong.")
