NAME

Clack.Handler.Hunchentoot - Clack handler for Hunchentoot.

SYNOPSIS

(defpackage clack-test  
  (:use :cl  
        :clack.handler.hunchentoot))  
(in-package :clack-test)  
 
;; Start Server  
(run (lambda (req)  
       '(200 nil ("ok")))  
     :port 8080)

DESCRIPTION

Clack.Handler.Hunchentoot is a Clack handler for Hunchentoot, Lisp web server. This package exports run' and stop'.

AUTHOR

Eitarow Fukamachi (e.arrows@gmail.com)

EXTERNAL SYMBOLS

  • Function: run [app &key debug (port 8080)]

    Start Hunchentoot server.

  • Function: stop [acceptor]

    Stop Hunchentoot server. If no acceptor given, try to stop `acceptor' by default.

  • Method: process-connection [(*acceptor* <debuggable-acceptor>) (socket t)]

  • Method: acceptor-request-dispatcher [(*acceptor* <debuggable-acceptor>)]
